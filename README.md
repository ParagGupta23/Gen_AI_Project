**LLM Engineering Journey**

This repository documents my hands-on work in building applications using local large language models through Ollama. The focus is on developing practical understanding and implementation skills across different stages of LLM-based systems.

**Objective**

The objective of this repository is to build end-to-end capabilities in LLM engineering by progressing from basic API usage to more advanced applications such as conversational systems, streaming interfaces, and retrieval-based solutions.

**Tech Stack**

Python
Ollama for local LLM execution
OpenAI compatible APIs
Requests and HTTP based integrations

**Projects**

01 Basic Ollama API
Implements a simple interaction with a locally hosted language model using an OpenAI compatible interface. This serves as the foundation for subsequent projects.

02 Chatbot with Memory
Focuses on building a conversational system that maintains context across multiple interactions by storing and managing message history.

03 Streaming Chat
Explores real-time response generation using streaming outputs to improve user experience in conversational interfaces.

04 Website and Document Summarizer
Aims to extract and summarize content from web pages or documents, forming the basis for retrieval augmented systems.

Key Learnings

Working with local LLM infrastructure
